Q&A


  1. Three main diffrences between web1, web2 and web3:
    • Pages: web1 has only static pages for advertisment of content, when web2 is dynamic and used for comunication between users, and web3 is even more dynamic and allows personal web expirience.
    • Search engine: web1 has a very simple search mechanism, when web2 has google, and web3 has advanced search and recommendation systems.
    • Abilitys: web1 was Read-only web, when web2 was an Read & Write web, and web3 is a Read, Write & Execute web.
  2. Use Case - Download contect from web:
    • Web1: download by pressing button that linked to a server holding the content, slow and unrelaiable.
    • Web2: download is integrated to the browser, safer and more relaiable, and also peer to peer download of contect.
    • Web3: running content on web instead of dowloading, personal cloud space that allows download from all OS platforms.
